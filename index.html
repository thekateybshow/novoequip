Uses GitHub Gist to store shared data. Free, no expiry, works on all devices.

Step 1: Go to github.com/settings/tokens/new
Step 2: Give it any name (e.g. "Equipment Tracker")
Step 3: Tick gist under scopes
Step 4: Click Generate token → copy it
Step 5: Paste below and tap Connect
